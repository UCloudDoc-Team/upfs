

# 产品规格
公测期目前只支持 412 MB/s/TiB 基线版本，如需其他规格请联系技术支持。


### 公式

每个集群的容量：6（机器数） * 8（每台机器上盘的数量）/ 3（三副本） * 6.4TB（每块盘的容量）= 102.4TB

每个集群的总带宽：50Gb/s * 6 = 300Gb/s

存储系统的带宽效率(每 TiB存储空间的每秒传输的兆字节数（MB/s）):300*1024/8/102.4/0.9095 = 412 MB/s/TiB
